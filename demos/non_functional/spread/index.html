Spreadsheet

An Excel-like spreadsheet powered by Wijmo/SpreadJS
Gets a live data stream from the Sliders Demo and can send data to the Gauges Demo

Open Sliders Demo Open Gauges Demo Source Code
To enter a formula, navigate to a cell, press backspace, and then type your formula, e.g. 
=SUM(A5:A12)
To subscribe to some topic, enter a formula 
=SUBSCRIBE("io.crossbar.demo.sliders.123456.master")
To publish to some topic, enter a formula 
=PUBLISH("io.crossbar.demo.gauges.g1", A3)
